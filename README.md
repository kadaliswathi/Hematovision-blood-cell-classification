# Hematovision-blood-cell-classification
Hematovision: Advanced Blood Cell Classification using Transfer Learning and Deep Learning. This project classifies different types of blood cells from microscopic images using CNN models to assist in medical analysis.
Blood Cell Classification using Deep Learning

This project classifies blood cells using deep learning and transfer learning.

## Technologies Used
- Python
- TensorFlow / Keras
- Flask
- HTML, CSS

## Features
- Upload blood cell image
- Predict blood cell type
- Display result on web page

## Project Description
This project uses a trained deep learning model to classify blood cell images into different categories. It helps automate medical image analysis.
